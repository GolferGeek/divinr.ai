# Effort: Landing Page

## Problem
Marketing copy exists (`docs/efforts/current/marketing-compilation/marketing-copy.md`) but there's no public-facing landing page to acquire users.

## Intention
Build a landing page from the compiled marketing materials: hero section, feature highlights, how-it-works flow, trust/transparency narrative, and call-to-action for signup.

## Scope
- Landing page at divinr.ai (public, no auth required)
- Hero: "AI market analysis you can actually understand"
- Feature sections from marketing copy (9 core + 2 social features)
- Explainability narrative section
- User persona journeys (optional)
- CTA → signup/invite flow
- Mobile responsive (375px–1440px)

## Out of Scope
- Pricing page (deferred to Stripe integration)
- Blog/content marketing
- SEO optimization (first pass only)
