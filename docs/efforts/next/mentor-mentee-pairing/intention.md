# Effort: Mentor/Mentee Pairing

## Problem

New users have no guidance beyond the AI analysts. Experienced members who've proven their skill have no way to help others or earn recognition for teaching. The learning curve is steep without human guidance.

## Intention

Build a mentor/mentee system within clubs where experienced members (high tournament performance, high learning score) can be paired with newer members. Mentors review mentee trades, share strategy journal entries, and guide them through prediction challenges.

## Scope

- Mentor eligibility: minimum tournament count, minimum win rate, club admin approval
- Mentee request: members can request a mentor within their club
- Pairing: admin matches mentors to mentees (1:3 max ratio)
- Mentor dashboard: see mentee trades, journal entries, challenge responses
- Mentee view: see mentor's public journal entries and tournament history
- DM channel auto-created between mentor and mentee
- Mentor recognition: "Mentor" badge, mentor leaderboard within club
- Feedback loop: mentee rates mentor helpfulness quarterly

## Dependencies
- Learning clubs must ship first
- Messaging system (for mentor-mentee DMs)
