# Tournament System — Marketing Blurb

Compete with friends on market analysis. Create a tournament, set a starting balance and time window, and invite others via a shareable link. Each player gets an isolated virtual portfolio — queue trades, manage positions, and watch the leaderboard update in real time. Weekly sprints keep it fast, while season-long events reward consistency. Every trade is paper-only, every portfolio is transparent, and every ranking is earned. No real money, no risk — just pure analytical competition.
