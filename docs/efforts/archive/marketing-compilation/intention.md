# Effort: Marketing Compilation

## Problem
Each testing effort produces a marketing blurb for its feature area. This effort compiles those blurbs into cohesive marketing materials.

## Intention
Take the marketing angles from all testing efforts and produce:
1. Landing page copy — the "why Divinr" story
2. Feature inventory — every feature with a one-liner and screenshot/demo reference
3. Differentiation narrative — "explainability over black boxes"
4. User personas and their journeys through the product

## Scope
- Compile marketing blurbs from all test-* efforts
- Write landing page sections (hero, features, how-it-works, trust/transparency)
- Create feature inventory document for promotional use
- Draft the explainability narrative
- Identify which features are most marketable vs. table-stakes

## Out of Scope
- Actual landing page HTML/design (implementation effort)
- Paid advertising strategy
- Video production
- Pricing page (that's power-user-expansion)
