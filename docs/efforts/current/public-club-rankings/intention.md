# Effort: Public Club Rankings

## Problem

Clubs exist in isolation. There's no way to see how your club compares to others, no competitive pressure between clubs, and no social proof for prospective members choosing which public club to join.

## Intention

Build cross-club leaderboards and rankings that create inter-club competition and help users discover the best-performing clubs. Rankings based on aggregate tournament performance, learning score, and member growth.

## Scope

- Global club leaderboard: ranked by aggregate return %, learning score, member count
- Club profile badges: "Top 10%", "Rising Club", "Most Improved"
- Club comparison view: side-by-side stats between two clubs
- Discovery sort/filter by ranking metrics
- Seasonal rankings (monthly/quarterly resets)

## Dependencies
- Learning clubs must ship first
